# Split Pane Navigation
move panel right: `ctrl+]`
move panel left: `ctrl+[`

focus right: `cmd+f cmd+r`
focus left: `cmd+f cmd+l`

open (split): `ctrl+enter` (from explorer menu)

# Emacs
`M` = `opt`
`C` = `ctrl`

`ctrl+v`: Scroll DOWN by one screen unit
`opt+v`: Scroll UP by one screen unit


# chords
`fold`: opt+cmd+[
`unfold`: opt+cmd+]

# Specs: cmd+k cmd+?
`a`: all specs
`e`: file (all)
`g`: focused spec
`v`: next failing spec
`z`: re-run last spec

# Go: cmd+k cmd+?
`g`: run test@cursor || re-run last spec
`h`: run all file tests
