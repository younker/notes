# Program Context

Synthesis of conversations with [[Joel Dohm]] (AD), [[Eric Liebler]] (program assistant, leaving), [[Noah Honaker]] (prior coach), [[Jenni Attie]] (parent/board), [[Zach Jonker]] (prior coach), and the [[20260519 Player Meeting]] agenda.

## Status

- Hired as varsity head coach (2026 season).
- [[Matt Hill]] hired as JV coach.
- Need to complete 9v9 grassroots course to qualify for the D license course ($600).

## Program State

- Numbers are tight: low 30s total, current freshman class only ~5 players; senior class is small; 4-5 graduated last year.
- Skill level varies widely across the roster.
- Conference: TC is toughest. Gaylord, Alpena, Cadillac usually competitive. Soo is a gimme. Elk Rapids (out of conference) and early-season opponents like Marquette can be tough.
- Account balance ~$3-4k. Charter bus runs ~$1800. [[Noah Honaker]] used yellow buses; [[Zach Jonker]] ran budget closer to zero.
- Funding historically not a barrier. Boosters cover concessions/fund raising. Families typically asked to donate ~$50 in lieu of fund-raising effort.

## Schedule and Travel

See [[Boys Soccer Fall 26]] for the game schedule.

- League games every Tuesday and Thursday - brutal cadence.
- Travel is heavy: depart ~1pm, return ~10pm on game days.
- Weekly rhythm that worked for [[Zach Jonker]]:
  - Sunday evening: 1h15 training
  - Monday: 90 min training
  - Tuesday: game
  - Wednesday: light recovery
  - Thursday: game
  - Friday: kids are spent - yoga / recovery only
- [[Noah Honaker]]'s view: schedule first practice of the week on Sunday; mid/late week is low energy; Friday is recovery (yoga). Stadium turf is tough on joints - split practices between stadium and Click field.
