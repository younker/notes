## 2. 2019-04-16 16:48 — GIS@dshs.wa.gov

*Subject: RE: Geospatial Open Data — Business Owner & Operator*

Hi Jason,

Two potential sources come to mind.

1.       From the DSHS Assisted Living Facilities metadata, you might contact the source data providers to ask if an API with the contact you’re seeking is available.

·         http://geo.wa.gov/datasets/WADSHS::dshs-assisted-living-facilities > View Metadata

·         Indicates this source:  https://fortress.wa.gov/dshs/adsaapps/lookup/BHAdvLookup.aspx > Contact ALTSA

2.       WA Secretary of State provides this business search capability.  You might contact them to ask about an API

·         https://ccfs.sos.wa.gov/#/AdvancedSearch

Finally, I’m aware that unavailability of that information in an API possibly could be related to Washington State law preventing agencies from releasing names of people if the use is for commercial purposes.

·         https://apps.leg.wa.gov/rcw/default.aspx?cite=42.56.070 (8)
