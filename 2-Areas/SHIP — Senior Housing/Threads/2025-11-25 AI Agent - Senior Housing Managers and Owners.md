## 1. 2025-11-25 20:35 — justin@cascadiadevelopment.com

Manager

  1.  DSHS List - https://fortress.wa.gov/dshs/adsaapps/lookup/BHAdvLookup.aspx select all, download CSV
  2.  If "has reports" is yes, then go to report URL in column AF, and gather all reports
  3.  Google each address to find manager company website to verify manager and gather more info on community and the operator
  4.  Once manager is identified, compile the data for that manager

Owner

  1.  Using address, look up county on CSV and find county website with owner info "king county parcel information"
  2.  Go to "launch Parcel Viewer" https://gismaps.kingcounty.gov/parcelviewer2/
  3.  Search by Address and input address into search bar and search "1845 116th Ave NE"
  4.  Get Taxpayer name "ASC OVERLAKE LLC"
  5.
