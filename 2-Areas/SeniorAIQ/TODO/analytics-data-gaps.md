# Analytics Data Gaps

## Problem

### 2,697 facilities with no Regulatory data

A large number of facilities have zero regulatory data. Is this expected (e.g., facilities that have never been inspected), a collection gap, or a data pipeline issue?

## Investigation

- For the 2,697 facilities with no regulatory data: what do they have in common? (county, facility type, license status, etc.)
- Is regulatory data only collected for certain facility types or counties?
- Are there DSHS pages for these facilities that have inspection data we're not collecting?

## Goal

Identify root cause for the regulatory data gap and determine whether it is expected, a collection bug, or missing functionality.
