# Facility Metrics Summary for Justin

## Task

Prepare a brief summary of each facility metric/scoring category for Justin. This should explain what each metric measures, where the data comes from, and how it contributes to the overall facility score.

## Categories to Cover

- Regulatory (DSHS inspections, complaints, enforcement actions)
- Reviews (Google reviews, sentiment)
- Property/Ownership (assessed value, property details)
- Location/POI (nearby hospitals, amenities)
- Leadership (staff, management)
- Brand (operator reputation, portfolio)

## Format

Keep it concise -- a few sentences per metric explaining what it is, what data feeds it, and what a high/low score means.
